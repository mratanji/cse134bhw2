The <li> element.

The <li> element is used to indicate elements in either a <ol> or <ul> element. It MUST be inside one in order ot validate correctly. For an example, look at the bullet points for <ol> and <ul> above.

The <dl>, <dt>, and <dd> elements

The <dl>, <dt>, and <dd> elements are description lists. If you have a <dl> element and either a <dt> or <dd> element nested in it, you MUST have both <dt> and <dd> elements, otherwise it is non-conforming. The correct version with all tags is:
