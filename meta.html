This page is testing the meta element, which has to be included in the head element. You can only have one meta element per page. In the meta element, you can specify keywords that pertain to your page, and also include a pragma directive. On this page, the pragma directive is to refresh every 5 seconds. You can also specify a name for the page, as well as the character encoding for the page. Only one of the charset, name, or http-equiv attributes can be specified.
